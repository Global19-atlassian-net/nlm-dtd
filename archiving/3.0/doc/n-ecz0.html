document-id

Document Identifier

The identifier of a document within a publication that is (or contains) the target of the <related-object> element

Remarks

Used along with @document-id-type to identify the document within a publication that is (or contains) the target of the <related-object> element. The @document-id might contain a DOI as content, and the @document-id-type should indicate that this ID is a DOI.

Attribute Values

In Element

<related-object>
Value Meaning
Text, numbers, or special characters An identifier for the document that contains the related object<related-object>
Restriction: This attribute may be specified if the element is used.
