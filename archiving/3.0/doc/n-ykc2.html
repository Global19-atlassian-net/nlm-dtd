style

Style (XHTML table model; MathML 2.0 Tag Set)

This attribute is used in several contexts, to express the concrete style or rendering to be applied to the element.

Remarks

When in MathML, the value should be as defined in the Mathematical Markup Language (MathML) 2.0 Tag Set.

When in an XHTML table or, the value should be as defined in the W3C XHTML Tag Set.

When on a Styled Special (Subject) Content element, the value may be as defined in the W3C XHTML Tag Set.

Attribute Values

In Elements

<caption>, <col>, <colgroup>, <styled-content>, <table>, <tbody>, <td>, <tfoot>, <th>, <thead>, <tr>
Value Meaning
Text, numbers, or special characters Identifies inline style markup (See W3C documentation for the XHTML table model.)
Restriction: This attribute may be specified if the element is used.

In Element

<mml:math>
Value Meaning
Text, numbers, or special characters See MathML documentation.
Restriction: This attribute may be specified if the element is used.
