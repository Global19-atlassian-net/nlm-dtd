axis

Axis (XHTML table model)

A method of categorizing table cells. The value of this attribute should be one or more category names.

See W3C documentation for the XHTML table model.

Remarks

This attribute was not widely supported as of December 2002.

Attribute Values

In Elements

<td>
<th>
Value Meaning
Text, numbers, or special characters Names the axis
Restriction: This attribute may be specified if the element is used.
