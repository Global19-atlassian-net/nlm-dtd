X - Generated Text/Punctuation Class Elements

Class containing a single element that will hold generated punctuation or other generatable text, for example, the commas or semicolons between keywords

Remarks

This textual material element has been created to 1) match existing XML vocabularies, 2) identify such material, so it can be processed (e.g., removed):, and 3) try to remove “look-and-feel” material from semantic material such as keywords.
