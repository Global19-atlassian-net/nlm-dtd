%simple-phrase;

Simple Phrase-Level Textual Elements

In the base Suite, this parameter entity names elements that may be used almost anywhere text is used, for example, inside most kinds of titles. In this Archiving Tag Set, this parameter entity has been overridden to allow any phrase-level elements.

Element

Remarks

In the base Suite, this parameter entity is slightly more complex and contains more elements than the %simple-text; parameter entity; this parameter entity would be used for potentially complex models.
