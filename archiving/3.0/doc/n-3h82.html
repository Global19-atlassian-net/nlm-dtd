%counts-model;

Counts Model

Content Model for the <counts> element, a container element holding all the metadata elements that are “counts” of other elements, e.g., the number of figures, the number of tables, the number of pages, etc., in an article

Content Model

<!ENTITY % counts-model "(fig-count?, table-count?, equation-count?, 
                          ref-count?, page-count?, word-count?)"     >
