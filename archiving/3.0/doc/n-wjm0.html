count

Count

Contains the number of (a count of) various objects, such as equations, figures, tables, etc., that are contained within an article

Attribute Values

In Elements

<equation-count>, <fig-count>, <page-count>, <ref-count>, <table-count>, <word-count>
Value Meaning
Number Although this Tag Set cannot limit this attribute beyond letters, numbers, and special characters, validation beyond the DTD (for example, schema validation or Schematron validation) may constrain this attribute to a counting number (positive integer), for example “15”.
Restriction: This attribute must be specified if the element is used.
