Span (XHTML table model)

The number of columns that will share the <col> or <colgroup> element’s attributes

Attribute Values

In Elements

<col>
<colgroup>
Value Meaning
A counting number How many columns will have the same attributes, e.g., “3”
Default value: 1 (The attributes are for this column only.)
