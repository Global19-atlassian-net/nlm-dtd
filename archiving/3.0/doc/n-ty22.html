seq

Sequence

Used to record the sequence number of an article within the type of sequence named by the element to which this attribute is attached: on a <volume> element, the sequence within the volume; on an <issue> element, the sequence within the issue; on an <fpage> element, the sequence within a single page; and on an <elocation-id> element, the electronic sequence.

Usage

For journals (such as continuous-makeup journals) where more than one article can start on a single page, this attribute can record a sequence number or letter to differentiate this article from the other articles on the same page. For example, the first article to start on page 62 might be given a sequence of “62a”; the sequence letter would be “a”, with a sequence letter “b” for a second article, etc.

Attribute Values

In Elements

<elocation-id>, <fpage>, <issue>, <volume>
Value Meaning
Text, numbers, or special characters The number of the article in a sequence of articles within a volume, issue, electronic compilation, or even on the same page. For example, for journals (such as continuous-makeup journals) with more than one article starting on the same page, this attribute indicates a sequence number or letter for this article, such as “a”, “b”, “c”; or “1”, “2”, “3”. For example, a value of “2” could mean “554-2”, page 554, second article.
Restriction: This attribute may be specified if the element is used.
