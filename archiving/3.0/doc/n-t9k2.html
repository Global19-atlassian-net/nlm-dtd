product-type

Type of Product

The type of product being reviewed, for example, a book, software package, journal article or journal issue, website, film, hardware component, service, etc.

Attribute Values

In Element

<product>
Value Meaning
Text, numbers, or special characters The type of product being reviewed, for example, “website” or “videogame”
Restriction: This attribute may be specified if the element is used.
