%citation.class;

Citation Class Elements

Definition

Element class created to hold the <citation> element, which is a reference to an external document, as used within, for example, the text of a paragraph

Remarks

This class has been made intentionally small, so that the citation element may be used in many places, but restricted overall.

Declaration

<!ENTITY % citation.class
                        "citation"                                   >
