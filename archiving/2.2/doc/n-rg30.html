<boxed-text>

Boxed Text

Definition

Textual material that is part of the body of text but outside the flow of the narrative text, for example, a sidebar, marginalia, text insert (whether enclosed in a box or not), caution, tip, note box, etc.

Attributes

content-type Type of Content
id Identifier
position Position
xml:lang Language

Related Elements

A footnote <fn> usually contains material that cannot stand alone; it must be related to a word, phrase, or paragraph already in the basic material. Supplementary text that stands on its own or that is related to but not directly tied to material in the text, can be tagged using the <boxed-text> element.

Model Information

Content Model

<!ELEMENT  boxed-text   %boxed-text-model;                           >

Description

The following, in order:

  • Object Identifier <object-id>, zero or more
  • The following, in order:
    • Section Metadata <sec-meta>, zero or one
    • Label (of a Figure, Reference, Etc.) <label>, zero or one
    • Title <title>, zero or one
    • Any combination of:
      • Array (Simple Tabular Array) <array>
      • Boxed Text <boxed-text>
      • Chemical Structure (Display) <chem-struct>
      • Chemical Structure Wrapper <chem-struct-wrapper>
      • Figure <fig>
      • Figure Group <fig-group>
      • Graphic <graphic>
      • Media Object <media>
      • Preformatted Text <preformat>
      • Supplementary Material <supplementary-material>
      • Table Wrapper <table-wrap>
      • Table Wrapper Group <table-wrap-group>
      • Formula, Display <disp-formula>
      • Paragraph <p>
      • Definition List <def-list>
      • List <list>
      • Tex Math Equation <tex-math>
      • Math (MathML 2.0 DTD) <mml:math>
      • Related Article Information <related-article>
      • Acknowledgments <ack>
      • Quote, Displayed <disp-quote>
      • Speech <speech>
      • Statement, Formal <statement>
      • Verse Form for Poetry <verse-group>
    • Section <sec>, zero or more
    • Any combination of:
      • Notes <notes>
      • Footnote Group <fn-group>
      • Glossary Elements List <glossary>
      • Reference List (Bibliographic) <ref-list>
  • Any combination of:
    • Attribution <attrib>
    • Copyright Statement <copyright-statement>
    • Permissions <permissions>

This element may be contained in:

<abstract> Abstract; <ack> Acknowledgments; <app> Appendix; <app-group> Appendix Group; <body> Body of the Article; <boxed-text> Boxed Text; <disp-quote> Quote, Displayed; <gloss-group> Glossary Group; <glossary> Glossary Elements List; <named-content> Named Special (Subject) Content; <notes> Notes; <p> Paragraph; <ref-list> Reference List (Bibliographic); <sec> Section; <trans-abstract> Translated Abstract
