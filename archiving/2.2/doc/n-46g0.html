<table>

Table (XHTML table model)

Definition

The rows and columns part of a full table. The standard XHTML table model has been used to take advantage of existing software for table handling. For more complete information on this table model, see the documentation for the HTML 4.0 (XHTML) table model. (Note: The <table> element is usually just part of what would be called a “table” in print; a full table may include a table caption, a table title, table footnotes, introductory paragraphs, etc., and is created using the <table-wrap> element.)

Remarks

The standard XHMTL model has been modified to remove the <caption> element from within the <table> element, because, in the modular Archiving and Interchange DTD Suite, the <caption> element has been made part of the <table-wrap> element.

Historical Note: CALS Tables — As part of the 0.1 version of this DTD, the OASIS Interchange (CALS) table model was also included in the Suite, to prove that it was possible, though not easy, to include two different table models, by using pseudo-namespaces to differentiate any name clashes. Since there were indeed name clashes, the XHTML model was taken as primary (no name spaces) and the OASIS Exchange CALS model was set up to use pseudo-namespaces. Although all mention of the CALS model has been removed from this DTD and the Suite, the modules to make the CALS model possible are still shipped with the base Suite. The following changes were made to that model: the %titles; Parameter Entity for the OASIS table model has been set to the null string (""), since the table title <title> is also part of the <table-wrap>. Other than the addition of namespace prefixes, no other changes were made to the OASIS table content model.

Related Elements

The <table> element is an element defined by a public table model such as the XHTML table model. This element is not, in itself, a full table, but only the rows and columns portions of a table. A full table may have other features, such as a title, caption, table footnotes, etc. The <table-wrap> element contains the full table. A full table may or may not include a <table> element within it, since other constructions, such as 2-part lists, may also be given table titles, captions, etc.

Model Information

Content Model

<!ELEMENT %table.qname;  %table.content; >

Description

The following, in order:

  • Any one of:
    • Table Column (XHTML table model) <col>, zero or more
    • Table Column Group (XHTML table model) <colgroup>, zero or more
  • Any one of:
    • The following, in order:
      • Table Header (XHTML table model) <thead>, zero or one
      • Table Footer (XHTML table model) <tfoot>, zero or one
      • Table Body (XHTML table model) <tbody>, one or more
    • Table Row (XHTML table model) <tr>, one or more
