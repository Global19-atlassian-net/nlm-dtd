seq

Sequence of Same-page Articles

Definition

Sequence number of an article that starts on the same page as one or more other articles

Usage

For journals (such as continuous-makeup journals) with more than one article starting on the same page, this attribute indicates a sequence number or letter for this article. For example, the first article to start on page 62 might be given a sequence of “62a”; the sequence letter would be “a”, with a sequence letter “b” for a second article, etc.

Attribute Values

In Elements

<elocation-id> Electronic Location Identifier
<fpage> First Page
Value Meaning
Text, numbers, or special characters The number of the article in a sequence of articles on the same page. Traditionally, journals use counting letters or numbers, such as “a”, “b”, “c”; or “1”, “2”, “3”. For example, a value of “2” could mean “554-2”, page 554, second article.
Restriction: This attribute may be specified if the element is used.
