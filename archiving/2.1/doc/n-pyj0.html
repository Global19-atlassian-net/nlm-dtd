Default Element Classes Module

Definition

External module in the Archiving and Interchange DTD Suite that sets up the default Parameter Entities and element class definitions that are used to create the element mixes and define the content models for the base Suite

Remarks

It is expected that some or all of the Parameter Entities declared in this module will be overridden in the DTD-specific custom classes module, %archivecustom-classes.ent;.
