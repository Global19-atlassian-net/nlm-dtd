%oasis-tablesetup.ent;

OASIS XML Table Setup Module

Definition

Note: Not used in the current Archiving DTD.

External module named in the Archiving and Interchange DTD Suite, but not used in the Journal Archiving and Interchange DTD, to set up the Parameter Entities needed by the OASIS (CALS) Table Exchange table model

Remarks

Authoring Note: The Journal Archiving DTD uses the XHTML table model, which is called by the %XHTMLtablesetup.ent; module which calls the %htmltable.dtd; module to add the HTML 4.0 (XHTML) table tags to a tagset. If the OASIS (CALS) table model is wanted, the %oasis-tablesetup.ent; module should be invoked, which calls the %oasis-exchange.ent; module to add the OASIS table tags to a tagset.
