%intable-para.class;

Inside a Table Paragraph Class Elements

Definition

A collection of the simpler of the paragraph-level elements, named in one place, for use inside the <table-wrap> element, as an alternative to a table tag

Remarks

Although, in theory, all tables are composed of rows and columns, some journal DTDs allow a <table> element to contain such components as a definition list, a paragraph, a bulleted list, etc., instead of rows and columns.

This Parameter Entity is used inside the Parameter Entity %inside-table-wrapper;, which names all the elements allowed in the middle of a <table-wrap>.

Declaration

<!ENTITY % intable-para.class
                        "disp-quote | speech | statement |
                         verse-group"                                >
