rid

Reference to an Identifier

Definition

Points to the identifier of the associated element as a means of linkage, for example, from a <term> to a <def>, or from a <bio> to an <contrib>

Attribute Values

In Elements

<overline-end> Overline End
<underline-end> Underline End
Value Meaning
An identifier (IDREF) Points to the existing identifier
Restriction: This attribute must be specified if the element is used.

In Elements

<aff> Affiliation; <author-notes> Author Note Group; <bio> Biography; <contract-num> Contract/Grant Number; <contract-sponsor> Contract/Grant Sponsor; <contrib> Contributor; <def> Definition List: Definition; <term> Definition List: Term; <xref> X(cross) Reference
Value Meaning
One or more identifiers (IDREFS) Points to one or more existing identifiers
Restriction: This attribute may be specified if the element is used.
