XHTML Table Model Module

Definition

External module for the XHTML table model; this module is named in the Module to Name the Modules (%modules.ent;) and then called from the XHMTL Table Setup Module (%XHTMLtablesetup.ent;) to include table elements in the tag set for the DTD.
