<note>

Note in a Reference List

Definition

Used to tag an element inside a bibliographic reference list that is not a citation (for example, end note material when such a note is placed within a reference list)

Remarks

Conversion Note: This element should be limited to conversion use only. When new reference lists are created, notes should not be intermingled with citations.

Attribute

id Identifier

Model Information

Content Model

<!ELEMENT  note         %note-model;                                 >

Description

The following, in order:

  • Label (of a Figure, Reference, Etc.) <label>, zero or one
  • Paragraph <p>, one or more

This element may be contained in:

<ref> Reference Item
