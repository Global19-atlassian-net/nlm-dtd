%inline-supplementary-material-atts;

Inline Supplementary Material Attributes

Definition

Attribute list for the <inline-supplementary-material> element, which is a reference to supplementary material such as a dataset or video set inline with text

Remarks

This element was created to accommodate a small number of DTDs and is expected to be very rarely used.
