<title>

Title

Definition

Heading or title for a structural element, such as a Section, Figure, Boxed Text, etc.

Related Elements

Not used for any of the titles of an article. For an article title within either the article’s metadata or a bibliographic citation, use the element Article Title <article-title>. Within the article’s metadata, all article titles, included translated ones, are contained within the grouping element Title Group <title-group>. Within a bibliographic reference (element Citation), the title for the source of the material (such as a book) uses the Source <source> element.

Model Information

Content Model

<!ELEMENT  title        (#PCDATA %struct-title-elements;)*           >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Font <font>
  • Horizontal Rule <hr>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Overline Start <overline-start>
    • Overline End <overline-end>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
    • Underline Start <underline-start>
    • Underline End <underline-end>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (MathML 2.0 DTD) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Line Break <break>
  • X - Generated Text and Punctuation <x>

This element may be contained in:

<abstract> Abstract; <ack> Acknowledgments; <app> Appendix; <app-group> Appendix Group; <author-comment> Author Comment; <author-notes> Author Note Group; <back> Back Matter; <bio> Biography; <boxed-text> Boxed Text; <caption> Caption of a Figure, Table, Etc.; <def-list> Definition List; <disp-quote> Quote, Displayed; <fn-group> Footnote Group; <gloss-group> Glossary Group; <glossary> Glossary Elements List; <kwd-group> Keyword Group; <list> List; <notes> Notes; <> Reference List (Bibliographic Reference List); <sec> Section; <statement> Statement, Formal; <table-wrap-foot> Table Wrapper Footer; <trans-abstract> Translated Abstract
