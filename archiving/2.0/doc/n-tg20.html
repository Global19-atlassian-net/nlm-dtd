<ack>

Acknowledgments

Definition

Textual material that names the parties which the author wishes to thank or recognize for their assistance in, for example, producing the work, funding the work, inspiring the work, or assisting in the research on which the work is based

Remarks

While most journal DTDs limit the Acknowledgments to inside Back Matter, the Archiving DTD allows <ack> at either the paragraph or the section level at many places within the body. Usage within the body of an article will probably be rare.

Attributes

content-type Type of Named Content

Description

The following, in order:

  • Label (of a Figure, Reference, Etc.) <label>, zero or one
  • Title <title>, zero or one
  • Any combination of:
    • Array (Simple Tabular Array) <array>
    • Boxed Text <boxed-text>
    • Chemical Structure (Display) <chem-struct>
    • Chemical Structure Wrapper <chem-struct-wrapper>
    • Figure <fig>
    • Figure Group <fig-group>
    • Graphic <graphic>
    • Media Object <media>
    • Preformatted Text <preformat>
    • Supplementary Material <supplementary-material>
    • Table Wrapper <table-wrap>
    • Table Wrapper Group <table-wrap-group>
    • Formula, Display <disp-formula>
    • Paragraph <p>
    • Definition List <def-list>
    • List <list>
    • Tex Math Equation <tex-math>
    • Math (MathML 2.0 DTD) <mml:math>
    • Related Article Information <related-article>
    • Acknowledgments <ack>
    • Quote, Displayed <disp-quote>
    • Speech <speech>
    • Statement, Formal <statement>
    • Verse Form for Poetry <verse-group>
  • Section <sec>, zero or more
  • Any combination of:
    • Notes <notes>
    • Footnote Group <fn-group>
    • Glossary Elements List <glossary>
    • Reference List (Bibliographic Reference List) <>

This element may be contained in:
