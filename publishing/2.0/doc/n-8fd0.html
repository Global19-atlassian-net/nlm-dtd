%mathmlsetup.ent;

MathML Setup Module

Definition

External module in the base DTD Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from the Publishing DTD to set up the namespsaces and Parameter Entities for the MathML elements

Remarks

This module then calls the MathML DTD to include the MathML elements in a tag set.

Declaration

<!ENTITY % mathmlsetup.ent 
                        PUBLIC
"-//NLM//DTD Archiving and Interchange DTD Suite MathML Setup Module v1.1 20031101//EN"
"mathmlsetup.ent"                                                    >
