%block-display.class;

Display Class Elements

Definition

Names graphical or other image-related elements in the base DTD Suite that cannot be used inline, but only as block elements; the display elements may occur within the text of a table cell or paragraph although they are typically at the same hierarchical level as a paragraph.

Declaration

<!ENTITY % block-display.class
                        "array | boxed-text | chem-struct |
                         chem-struct-wrapper | fig | fig-group | 
                         graphic | media | preformat | 
                         supplementary-material | table-wrap |
                         table-wrap-group"                           >

Module
