xmlns:xlink

XLink Namespace Declaration

Definition

This is not an attribute, but the namespace pseudo-attribute. The value provides a prefix to use for the XLink linking attributes. All namespace prefixes must be associated with a URL, and the prefix “xlink” has been set to the URL to the World Wide Web Consortium (W3C) XLink Recommendation.

Attribute Values

In Elements

<abbrev> Abbreviation or Acronym; <article> Article; <bio> Biography; <chem-struct> Chemical Structure (Display); <citation> Citation; <collab> Collaborative (Group) Author; <contract-num> Contract/Grant Number; <contract-sponsor> Contract/Grant Sponsor; <contrib> Contributor; <email> Email Address; <ext-link> External Link; <graphic> Graphic; <inline-graphic> Graphic, Inline; <inline-supplementary-material> Inline Supplementary Material; <institution> Institution Name: in an Address; <license> License Information; <media> Media Object; <named-content> Named Special (Subject) Content; <nlm-citation> NLM Citation Model; <product> Product Information; <related-article> Related Article Information; <self-uri> URI for This Same Article Online; <supplementary-material> Supplementary Material; <uri> Uniform Resource Indicator (URI)
Value Meaning
http://www.w3.org/1999/xlink The prefix “xlink” has been set to this URL, the URL of the W3C XLink Recommendation.
Restriction: This attribute value may not be changed.
