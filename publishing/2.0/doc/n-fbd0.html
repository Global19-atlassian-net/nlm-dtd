%phrase.ent;

Phrase-Level Content Elements Module

Definition

External module in the base DTD Suite that defines the Phrase Class Elements elements that is, names the inline, subject-specific elements

Remarks

At the time of this DTD’s creation, there were only two phrase-level elements. If more specific subject words (such as “gene”) are added to later versions of this DTD, they should be added to the %phrase.class; entity and defined in this module.

<!ENTITY % phrase.ent
