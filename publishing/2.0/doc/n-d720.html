<aff>

Affiliation

Definition

Name of a institution or organization, such as a university or corporation, that is the affiliation for a contributor such as an author or an editor

Remarks

Conversion Note: In a typical case, the id attribute of an <aff> element will be pointed to by one or more Contributor <contrib> elements.

Conversion Note: Even if the explicitly tagged numbers or symbols for author linkages are preserved during conversion, the linkage from a contributor to an affiliation should be made using the ID/IDREF mechanism during the conversion. The ID/IDREF should be created even if the mechanism must be inserted.

Attributes

id Identifier
rid Reference to an Identifier

Model Information

Content Model

<!ELEMENT  aff          (#PCDATA %aff-elements;)*                    >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address elements:
    • Address Line <addr-line>
    • Country: in an Address <country>
    • Fax Number: in an Address <fax>
    • Institution Name: in an Address <institution>
    • Phone Number: in an Address <phone>
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Line Break <break>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Label (Of a Figure, Reference, Etc.) <label>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>

This element may be contained in:

<article-meta> Article Metadata; <contrib> Contributor; <contrib-group> Contributor Group; <person-group> Person Group for a Cited Publication
