%article-meta-model;

Article Metadata Model

Definition

Full content model for the metadata that is specific to the article. Note: This includes both metadata that directly describes the article (such as the article title, abstract, or article ID), as well as the metadata that many MAJOUR-style DTDs call the issue metadata (such as the issue number and title)

Remarks

MAJOUR-style DTDs split the metadata into three categories: journal metadata, article metadata, and issue metadata. In this DTD Suite, the related element <journal-meta> holds the metadata specific to the journal.
