journal-id

Journal Identifier of a Related Article

Definition

The identifier for the journal which contains the related article

Usage

The journal-id attribute is used, along with the vol, issue, and page attributes, to provide the metadata for the related article.

Attribute Values

In Element

<related-article> Related Article Information
Value Meaning
Text, numbers, or special characters A journal abbreviation or other journal identifier
Restriction: This attribute may be specified if the element is used.
