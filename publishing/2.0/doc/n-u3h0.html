%XHTMLtablesetup.ent;

XHTML Table Setup Module

Definition

External module in the base DTD Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from a DTD to set up the the Parameter Entities needed by the HTML 4.0 (XHTML) table model

Remarks

Authoring Note: The Journal Publishing DTD uses the XHTML table model, which is called by the %XHTMLtablesetup.ent; module, which calls the %htmltable.dtd; module to add the HTML 4.0 (XHTML) table tags to a tagset.

<!ENTITY % XHTMLtablesetup.ent
