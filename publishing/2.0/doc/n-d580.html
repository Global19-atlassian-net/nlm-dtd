<disp-quote>

Quote, Displayed

Definition

Extract or extended quoted passage from another work, usually made typographically distinct from surrounding text

Remarks

Conversion Note: Use this element for epigraphs, as well as for block quotes and extracts within text.

Attributes

id Identifier
xml:lang Language

Model Information

Content Model

<!ELEMENT  disp-quote   %disp-quote-model;                           >

Description

The following, in order:

  • Title <title>, zero or one
  • Any combination of:
    • Array (Simple Tabular Array) <array>
    • Boxed Text <boxed-text>
    • Chemical Structure (Display) <chem-struct>
    • Chemical Structure Wrapper <chem-struct-wrapper>
    • Figure <fig>
    • Figure Group <fig-group>
    • Graphic <graphic>
    • Media Object <media>
    • Preformatted Text <preformat>
    • Supplementary Material <supplementary-material>
    • Table Wrapper <table-wrap>
    • Table Wrapper Group <table-wrap-group>
    • Formula, Display <disp-formula>
    • Paragraph <p>
    • Definition List <def-list>
    • List <list>
    • Tex Math Equation <tex-math>
    • Math (MathML 2.0 DTD) <mml:math>
    • Related Article Information <related-article>
    • Quote, Displayed <disp-quote>
    • Speech <speech>
    • Statement, Formal <statement>
    • Verse Form for Poetry <verse-group>
  • Any combination of:
    • Attribution <attrib>
    • Copyright Statement <copyright-statement>

This element may be contained in:

<app> Appendix; <app-group> Appendix Matter; <body> Body of the Article; <boxed-text> Boxed Text; <disp-quote> Quote, Displayed; <fig> Figure; <gloss-group> Glossary Group; <glossary> Glossary Elements List; <named-content> Named Special (Subject) Content; <notes> Notes; <p> Paragraph; <ref-list> Reference List (Bibliographic Reference List); <sec> Section; <supplementary-material> Supplementary Material; <table-wrap> Table Wrapper
