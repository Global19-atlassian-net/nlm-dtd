mime-subtype

Mime Subtype

Definition

The mime subtype for a graphic, multimedia object, or piece of supplementary material

Remarks

The source for this attribute’s values is the Internet Assigned Numbers Authority (IANA 2003).

Attribute Values

In Elements

<graphic> Graphic
<inline-graphic> Graphic, Inline
Value Meaning
cgm Computer Graphics Metafile
gif CompuServe Graphic Interchange Format
jpeg Joint Photographic Experts Group raster
png Portable Network Graphics
tiff Tag Image File Format
Restriction: This attribute may be specified if the element is used.

In Elements

<inline-supplementary-material> Inline Supplementary Material
<media> Media Object
<supplementary-material> Supplementary Material
Value Meaning
Text, numbers, or special characters A specific mime subtype (from the November 2003 list)
Restriction: This attribute may be specified if the element is used.
