%given-names-atts;

Given Names Attributes

Definition

Attribute list for the <given-names> element

Remarks

This Parameter Entity is currently set as a null string, i.e., there is no string content. However, a customization module could redefine this Parameter Entity, thereby adding new attributes to the <given-names> element.

For example, in the Journal Archiving DTD (Green), this Parameter Entity is used to add an attribute “initials to hold initials in those cases where they could not be algorithmically determined.

<!ENTITY % given-names-atts
             ""                                                      >
