%issue-title-elements;

Issue Title Elements

Definition

Elements that can be included along with data characters inside the content model of the <issue-title> element

Remarks

This Parameter Entity is currently set as a null string, i.e., there is no string content. However, a customization module could redefine this Parameter Entity, thereby adding elements to the <issue-title> element.

Declaration

<!ENTITY % issue-title-elements 
                        ""                                           >
