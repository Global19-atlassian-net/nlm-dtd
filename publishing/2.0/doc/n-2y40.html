<td>

Table Data Cell (XHTML table model)

Definition

See documentation for the HTML 4.0 (XHTML) table model.

One ordinary cell in a table, as opposed to a cell in the table header

Attributes

abbr Abbreviation (XHTML table model)
align Horizontal Alignment (XHTML table model)
alternate-form-of Alternate Form of Graphic, Media Object, Etc.
axis Axis (XHTML table model)
char Table Cell Alignment Character (XHTML table model)
charoff Table Character Offset (XHTML table model)
colspan Column Span (XHTML table model)
content-type Type of Named Content
headers Headers (XHTML table model)
id Identifier
rowspan Row Span (XHTML table model)
scope Scope (XHTML table model)
valign Vertical Alignment (XHTML table model)

Related Elements

There are two cells elements in the XHTML table model that contain the contents of the table: <td> contains ordinary data, and <th> contains table header data.

Model Information

Content Model

<!ELEMENT %td.qname;  %td.content; >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • Horizontal Rule <hr>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Formula, Display <disp-formula>
  • Line Break <break>
  • Citation <citation>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Definition List <def-list>
  • List <list>
  • Tex Math Equation <tex-math>
  • Math (MathML 2.0 DTD) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • Array (Simple Tabular Array) <array>
  • Chemical Structure (Display) <chem-struct>
  • Graphic <graphic>
  • Media Object <media>
  • Preformatted Text <preformat>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
