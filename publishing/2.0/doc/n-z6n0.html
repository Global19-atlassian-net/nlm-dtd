%date-types;

History Date Types

Definition

Names the values that may be used with the date-type attribute of the <date> element. This attribute should only be used if the date is one of the known types.

Remarks

The <history> element contains one or more <date> elements that record events in the life of a publication. Values for the date-type attribute include: “accepted” (the date the manuscript was accepted), “received” (the date manuscript received), “rev-request” (the date revisions were requested or manuscript was returned), and “rev-recd” (the date revised manuscript was received).

Declaration

<!ENTITY % date-types   "accepted | received | rev-request | 
                         rev-recd"                                   >
