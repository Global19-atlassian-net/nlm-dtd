%label.class;

Label Class

Definition

Class comprising the <label> element, which can be used to hold the number or symbol of a labeled object such as a footnote (“3”) or a table (“Table 6”)

Remarks

The Label Class Parameter Entity was created as a means of constraining use of the <label> element in the Publishing and Authoring DTDs.

Declaration

<!ENTITY % label.class  "label"                                      >

Module
