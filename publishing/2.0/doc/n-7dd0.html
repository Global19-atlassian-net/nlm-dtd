%nlmcitation.ent;

NLM Citation Module

Definition

External module in the Journal Publishing DTD (Blue) that contains the only new element created for the Journal Publishing DTD, the highly structured NLM citation (<nlm-citation>), to enforce a slightly loose version of an NLM-structured bibliographic reference. Sequence of elements is enforced and interior punctuation is not to be put into the data because the expectation is that it will be generated.

Declaration

<!ENTITY % nlmcitation.ent 
                        PUBLIC  
"-//NLM//DTD NLM Citation v2.0 20040830//EN"
"nlmcitation.ent"                                                    >
