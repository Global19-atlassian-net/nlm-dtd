%simple-intable-display.class;

Simple Table Display Elements

Definition

A very similar construction to the %simple-display.class; Parameter Entity, but with the Array (Simple Tabular Array) element excluded since an array is not permitted within a Table Wrapper

Declaration

<!ENTITY % simple-intable-display.class
                        "chem-struct | graphic | media | preformat"  >
