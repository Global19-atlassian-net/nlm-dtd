issue

Issue Number of a Related Article

Definition

The issue number of a journal which contains the related article

Usage

The issue attribute is used, along with the vol, journal-id, and page attributes, to provide the metadata for the related article.

Remarks

The issue number in which the primary journal article has been published is given in the <issue> element, which is part of the article metadata. This attribute is for related articles.

Attribute Values

In Element

<related-article> Related Article Information
Value Meaning
Text, numbers, or special characters The number of the journal issue, such as “48”
Restriction: This attribute may be specified if the element is used.
