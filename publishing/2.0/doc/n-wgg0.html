%address-link.class;

Address Link Class Elements

Definition

Link elements that can be used inside addresses. This is essentially the three generic external links: <ext-link>, <email>, and <uri>.

Remarks

In earlier releases, this Parameter Entity was named %address-elements; although it functioned as a class.

Declaration

<!ENTITY % address-link.class   
                        "email | ext-link | uri"                     >

Module

default-classes.ent
