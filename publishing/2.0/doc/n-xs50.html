<copyright-statement>

Copyright Statement

Definition

Copyright notice or statement, suitable for printing or display

Related Elements

The copyright year is usually identified within the copyright statement, as this statement is expected to be displayed. The year may also be repeated in the element <copyright-year>.

Model Information

Content Model

<!ELEMENT  copyright-statement
                        (#PCDATA %copyright-statement-elements;)*    >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Subscript <sub>
  • Superscript <sup>

This element may be contained in:

<array> Array (Simple Tabular Array); <article-meta> Article Metadata; <boxed-text> Boxed Text; <chem-struct-wrapper> Chemical Structure Wrapper; <disp-quote> Quote, Displayed; <fig> Figure; <graphic> Graphic; <media> Media Object; <supplementary-material> Supplementary Material; <table-wrap> Table Wrapper; <table-wrap-foot> Table Wrapper Footer; <verse-group> Verse Form for Poetry
