%abbrev-journal-title-elements;

Abbreviated Journal Title Elements

Definition

The elements that can be included along with data characters inside the content model of a <abbrev-journal-title> element

Remarks

This Parameter Entity is currently set as a null string, i.e., there is no string content. However, a customization module could redefine this Parameter Entity, thereby adding elements to the <abbrev-journal-title> element.

Declaration

<!ENTITY % abbrev-journal-title-elements
                        ""                                           >

Module
