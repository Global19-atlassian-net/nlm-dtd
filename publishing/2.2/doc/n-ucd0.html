%etal-model;

Et Al Model

Definition

The content model for the <etal> element

Remarks

Unlike the Journal Archiving DTD, in this Publishing DTD, <etal> has no content, but instead acts as an EMPTY placeholder for text which will be generated, such as “et al.”.

Declaration

<!ENTITY % etal-model   "EMPTY"                                      >
