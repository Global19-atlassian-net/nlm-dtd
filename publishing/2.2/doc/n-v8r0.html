%journalpubcustom-modules.ent;

Module to Name DTD-Specific Modules

Definition

External module in the Journal Publishing DTD (Blue) that names all Publishing-DTD-specific Suite external modules to be used by this DTD. (Note: This module does not declare itself, the DTD, any of the base DTD Suite modules, or the Suite Module of Modules which declares the base DTD Suite modules.)

"-//NLM//DTD Journal Publishing DTD-Specific Modules v2.2 20060430//EN"
