dtd-version

Version of the DTD

Definition

Identifies which version of the DTD, named in the DOCTYPE declaration, was used to make the article under consideration, which therefore can be used to validate this article

Remarks

Authoring Note: This is a “fixed” attribute (meaning that the value is set via a Parameter Entity in the DTD; thus, an author or editor cannot change it) that should change every time the DTD changes versions or revisions.

Attribute Values

In Element

<article> Article
Value Meaning
2.2 The 2.2 version of the Journal Publishing DTD is being used.
Restriction: This attribute value may not be changed.
