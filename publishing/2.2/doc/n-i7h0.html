%sec-model;

Content Model for a Structural Section

Definition

The model for a section-like structure that requires an initial title be present; this model is used for elements such as <sec> and <app>.

Declaration

<!ENTITY % sec-model    "(sec-meta?, label?, title, (%para-level;)*,
                          (%sec-level;)*, 
                          (%sec-back-matter-mix;)* )"                >
