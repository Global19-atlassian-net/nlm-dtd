<notes>

Notes

Definition

A section-like container element for a section of notes that may appear, for example, at the end of an <article> or as part of the article back matter

Remarks

A typical end-of-article note is a “Note in Proof”. A Note in Proof contains late-breaking news items or other material produced while the article was being typeset or was otherwise in production, that therefore happened too late to be included in the text of the article. This is typically NOT peer-reviewed content, and citations to other material within such content are usually integrated into the text or put inside the last paragraph, not listed separately in a bibliographic reference list <ref-list>.

Attributes

id Identifier
notes-type Type of Notes

Related Elements

Not to be confused with the general notes and footnotes that can appear at the bottom of a table (<table-wrap>), these should be tagged as ordinary paragraphs and footnotes inside a <table-wrap-foot> container element.

Model Information

Content Model

<!ELEMENT  notes        %notes-model;                                >

Description

The following, in order:

  • Section Metadata <sec-meta>, zero or one
  • Label (Of a Figure, Reference, Etc.) <label>, zero or one
  • Title <title>, zero or one
  • Any combination of:
    • Array (Simple Tabular Array) <array>
    • Boxed Text <boxed-text>
    • Chemical Structure (Display) <chem-struct>
    • Chemical Structure Wrapper <chem-struct-wrapper>
    • Figure <fig>
    • Figure Group <fig-group>
    • Graphic <graphic>
    • Media Object <media>
    • Preformatted Text <preformat>
    • Supplementary Material <supplementary-material>
    • Table Wrapper <table-wrap>
    • Table Wrapper Group <table-wrap-group>
    • Formula, Display <disp-formula>
    • Paragraph <p>
    • Definition List <def-list>
    • List <list>
    • Tex Math Equation <tex-math>
    • Math (MathML 2.0 DTD) <mml:math>
    • Related Article Information <related-article>
    • Quote, Displayed <disp-quote>
    • Speech <speech>
    • Statement, Formal <statement>
    • Verse Form for Poetry <verse-group>
  • Section <sec>, zero or more
  • Any combination of:
    • Footnote Group <fn-group>
    • Glossary Elements List <glossary>
    • Reference List (Bibliographic Reference List) <ref-list>

This element may be contained in:

<back> Back Matter; <front> Front Matter; <journal-meta> Journal Metadata
