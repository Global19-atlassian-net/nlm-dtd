<trans-abstract>

Translated Abstract

Definition

An abstract that has been translated into a language other than that in which the article title was originally published

Attributes

abstract-type Type of Abstract
xml:lang Language

Related Elements

See element Abstract <abstract>, which is the original-language version of the abstract.

Model Information

Content Model

<!ELEMENT  trans-abstract
                        %trans-abstract-model;                       >

Description

The following, in order:

  • Title <title>, zero or one
  • Paragraph <p>, zero or more
  • Section <sec>, zero or more
