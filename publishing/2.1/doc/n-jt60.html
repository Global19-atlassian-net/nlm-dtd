%dtd-version;

DTD Version

Definition

The version of the Journal Publishing DTD that was used to make the document instance under consideration

Remarks

Note that this is a fixed value that should only be changed when the DTD changes versions or revisions, and must be changed in the DTD. Like attributes with defaults, the value of the attribute is assigned to a document during parsing or other XML processing. The dtd-version attribute cannot be changed in a document instance.

Declaration

<!ENTITY % dtd-version
            "dtd-version
                        CDATA                          #FIXED '2.1'" >
