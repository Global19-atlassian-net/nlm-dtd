contrib-type

Type of Contribution

Definition

Identifies the contribution of an individual, for example, as an author, editor, contributor, translator, illustrator, designer, research assistant, etc.

Attribute Values

In Element

<contrib> Contributor
Value Meaning
Text, numbers, or special characters Word or phrase naming or describing the contribution. Typical values include “author” and “editor”.
Restriction: This attribute may be specified if the element is used.
