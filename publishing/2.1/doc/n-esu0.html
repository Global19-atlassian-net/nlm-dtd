<long-desc>

Long Description

Definition

Full description or summary of the content of a graphical object, table, or textual object such as a text box. The long description is used by some systems to make the object accessible, even to people or systems that cannot read/see/display the object.

Remarks

Conversion Note: This element is not to be used as a replacement for <caption>, which is a visual element displayed as part of a figure, table, etc. The <long-desc> is not a visual element; rather, it is used to be spoken in circumstances where the graphical element cannot be viewed. A similar element <alt-text> is also not usually visual, but instead is short and may be displayed when the figure, caption, or other major element is not available or cannot be processed by the person or device type being addressed, for example, “behind” the graphic on a website.
