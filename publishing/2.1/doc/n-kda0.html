<inline-formula>

Formula, Inline

Definition

A mathematical equation, expression, or formula that is to be displayed inline. The mathematics itself can be expressed as ASCII characters, as a graphic, or using TeX, LaTeX, or MathML mathematics expressions.

Remarks

Some print-oriented DTDs use a specific element to indicate “go into math mode or math font here”; while that was not the intended usage of this element, in nearly all cases the text inside such math-changes is a mathematical expression and it would not be incorrect to use this element to mark such material.

Attributes

alternate-form-of Alternate Form of Graphic, Media Object, Etc.
id Identifier

Related Elements

For a mathematical equation, expression, or formula which is to be displayed as a block (callout), use the Formula, Display <disp-formula> element.

Model Information

Content Model

<!ELEMENT  inline-formula
                        %inline-formula-model;                       >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (MathML 2.0 DTD) <mml:math>
  • Subscript <sub>
  • Superscript <sup>
