license-type

Type of License

Definition

Used on a <license> element to indicate the type of license involved

Attribute Values

In Element

<license> License Information
Value Meaning
Text, numbers, or special characters The type of license, for example, “open-access” for a license permitting unrestricted use, distribution, and reproduction in any medium, provided the original work is properly cited
Restriction: This attribute may be specified if the element is used.
