<title>

Title

Definition

Heading or title for a structural element, such as a <sec>, <app>, <boxed-text>, etc.

Related Elements

In spite of its name, this element is not used for any of the titles of an article:

  • Within the metadata for the article, the titles (which are grouped in a <title-group> wrapper) are explicitly named: <article-title>, article <subtitle>, article translated title <trans-title>, translated subtitle <trans-subtitle>, and article alternate title <alt-title>; and
  • Within the bibliographic reference (<citation> or <nlm-citation>) an article-title is also <article-title>, while a book or journal title (that contains the cited article or chapter) is <source>.

Model Information

Content Model

<!ELEMENT  title        (#PCDATA %struct-title-elements;)*           >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (MathML 2.0 DTD) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Line Break <break>

This element may be contained in:

<abstract> Abstract; <ack> Acknowledgments; <app> Appendix; <app-group> Appendix Matter; <author-comment> Author Comment; <author-notes> Author Note Group; <back> Back Matter; <bio> Biography; <boxed-text> Boxed Text; <caption> Caption of a Figure, Table, Etc.; <def-list> Definition List; <disp-quote> Quote, Displayed; <fn-group> Footnote Group; <gloss-group> Glossary Group; <glossary> Glossary Elements List; <kwd-group> Keyword Group; <list> List; <notes> Notes; <ref-list> Reference List (Bibliographic Reference List); <sec> Section; <statement> Statement, Formal; <table-wrap-foot> Table Wrapper Footer; <trans-abstract> Translated Abstract
