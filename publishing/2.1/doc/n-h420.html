<addr-line>

Address Line

Definition

An address <address> is a postal address and/or other contact information. An address line is one physical or logical line of an address, typically containing undifferentiated portions of an <address> such as street addresses, post box numbers, building numbers, mailstops, etc. They are called “lines” because addresses are frequently divided onto multiple lines and this encoding allows that fact to be preserved.

Remarks

There are many ways to tag an address. An address may be tagged merely as text; as a series of named address elements including <country>, <email>, <fax>; as a series of address lines <addr-line> which may or may not have other elements such as <country> identified within them.; as a mixture of named elements and address lines, etc.

Conversion Note: If the address is divided into lines, preserve them using this element. If the address is just a block of undifferentiated text, numbers, or special characters, the entire address may be placed inside one <addr-line> element.

Related Elements

The elements used in an address include the address class elements: <addr-line>, <country>, <fax>, <institution>, and <phone> and the address linking class elements: <email>, <ext-link>, and <uri>.

Model Information

Content Model

<!ELEMENT  addr-line    (#PCDATA %addr-line-elements;)*              >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • Subscript <sub>
  • Superscript <sup>

This element may be contained in:

<address> Address/Contact Information; <aff> Affiliation; <corresp> Correspondence Information
