%etal-elements;

Et Al Elements

Definition

Elements that can be included along with data characters inside the content model of an <etal> element. Note that in this DTD this Parameter Entity is not used, since the content model for <etal> is the keyword EMPTY.

Remarks

This Parameter Entity is currently set as %just-rendition;, which in turn is a null string, i.e., there is no string content. However, a customization module could redefine this Parameter Entity, thereby adding elements to the <etal> element.

Declaration

<!ENTITY % etal-elements
                        "%just-rendition; "                          >

Module

common.ent
