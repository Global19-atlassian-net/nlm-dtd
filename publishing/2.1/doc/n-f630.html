<colgroup>

Table Column Group (XHTML table model)

Definition

See documentation for the HTML 4.0 (XHTML) table model.

A container element to hold column descriptions (element <col>) for a table. May describe the default behavior for all columns with its attributes.

Attributes

align Horizontal Alignment (XHTML table model)
alternate-form-of Alternate Form of Graphic, Media Object, Etc.
char Table Cell Alignment Character (XHTML table model)
charoff Table Character Offset (XHTML table model)
content-type Type of Content
id Identifier
span Span (XHTML table model)
valign Vertical Alignment (XHTML table model)
width Width (XHTML table model; MathML 2.0 DTD)

Model Information

Content Model

<!ELEMENT %colgroup.qname;  %colgroup.content; >

Description

Table Column (XHTML table model) <col>, zero or more

This element may be contained in:

<table> Table (XHTML table model)
