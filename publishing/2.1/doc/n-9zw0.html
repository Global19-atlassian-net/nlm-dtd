<prefix>

Prefix

Definition

Honorifics or other qualifiers that usually precede a person’s name, for example, “Professor”, “Rev.”, “President”, “Senator”, “Dr.”, “Sir”, “The Honorable”, etc.

Model Information

Content Model

<!ELEMENT  prefix       (#PCDATA %prefix-elements;)*                 >

Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<name> Name of Person; <speaker> Speaker
