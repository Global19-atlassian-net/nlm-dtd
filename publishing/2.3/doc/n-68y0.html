%pub-id-types;

Article/Publication Identifier Types

Definition

The explicit list of values for the pub-id-type attribute, which names the type of identifier, or the organization or system that defined the identifier, for a specific ID of a journal article or other work

Remarks

The pub-id-type attribute names the type of identifier, or the organization or system that defined the identifier, for the identifier of the journal article or a cited publication. This attribute is used on the <article-id> element, which holds an identifier for the entire article. Among other elements using this attribute is the <pub-id> element, which is an identifier for a publication cited in a bibliographic reference (<citation> or <nlm-citation>).

Declaration

<!ENTITY % pub-id-types  "art-access-id | coden | doaj | doi |
                          manuscript | medline | other | pii |
                          pmcid |  pmid | publisher-id | sici"       >
