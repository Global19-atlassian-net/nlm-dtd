%nlmcitation.ent;

NLM Citation Module

Definition

External module in the Journal Publishing DTD (Blue) that contains the only new element created for the Journal Publishing DTD, the highly structured NLM citation (<nlm-citation>). This element enforces a slightly loose version of an NLM-structured bibliographic reference in which sequence of elements is enforced and interior punctuation should not be in the data because the expectation is that such punctuation will be generated.

Declaration

<!ENTITY % nlmcitation.ent
                        PUBLIC
"-//NLM//DTD NLM Citation v2.3 20070202//EN"
"nlmcitation.ent"                                                    >
