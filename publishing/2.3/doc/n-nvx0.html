<permissions>

Permissions

Definition

Wrapper element to hold copyright information, license material, and any future similar metadata

Model Information

Content Model

<!ELEMENT  permissions
                        %permissions-model;                          >

Description

The following, in order:

  • Copyright Statement <copyright-statement>, zero or one
  • Copyright Year <copyright-year>, zero or one
  • Copyright Holder <copyright-holder>, zero or one
  • License Information <license>, zero or one

This element may be contained in:

<app> Appendix; <array> Array (Simple Tabular Array); <article-meta> Article Metadata; <boxed-text> Boxed Text; <chem-struct-wrapper> Chemical Structure Wrapper; <disp-quote> Quote, Displayed; <fig> Figure; <front-stub> Stub Front Metadata; <graphic> Graphic; <media> Media Object; <preformat> Preformatted Text; <sec-meta> Section Metadata; <statement> Statement, Formal; <supplementary-material> Supplementary Material; <table-wrap> Table Wrapper; <table-wrap-foot> Table Wrapper Footer; <verse-group> Verse Form for Poetry
