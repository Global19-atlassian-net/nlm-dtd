<phone>

Phone Number: in an Address

Definition

Portion of an <address> which contains a phone number

Attribute

content-type Type of Content

Related Elements

The elements used in an address may include the address class elements: <addr-line>, <country>, <fax>, <institution>, and <phone> and the address linking class elements: <email>, <ext-link>, and <uri>.

Model Information

Content Model

<!ELEMENT  phone        (#PCDATA %phone-elements;)*                  >

Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<address> Address/Contact Information; <aff> Affiliation; <corresp> Correspondence Information
