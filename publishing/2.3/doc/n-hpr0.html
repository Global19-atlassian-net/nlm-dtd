%fn-atts;

Footnote Attributes

Definition

Attribute list for the <fn> element

Remarks

In this Publishing (Blue) Tag Set, unlike in the Journal Archiving (Green) Tag Set, specific values have been provided for the fn-type attribute by way of the %fn-types; Parameter Entity.

Declaration

<!ENTITY % fn-atts
             "id        ID                                 #IMPLIED
              fn-type   (%fn-types;)                       #IMPLIED
              symbol     CDATA                             #IMPLIED
              xml:lang  NMTOKEN                            #IMPLIED" >
