<journal-subtitle>

Journal Subtitle (Full)

The subtitle of the journal in which the article is published; part of the journal metadata

Remarks

This element (<journal-subtitle>) has been significantly remodeled in version 3.0 of the Tag Set. The values and/or usage in version 3.0 are not backward-compatible with that in previous versions of the Tag Set. Specifically, in prior versions the <journal-subtitle> element was allowed outside the (new) <journal-title-group> container element.
