%def-list-atts;

Default Definition List Attributes

Attribute list for the element <def-list> element, which describes 2-part lists (definition lists)

Remarks

This parameter entity is currently set as a null string, i.e., there is no string content. However, a customization module could redefine this parameter entity, thereby adding new attributes to the <def-list> element.

The attribute list for Definition List also includes the attributes defined by the Default List Class Attributes parameter entity
