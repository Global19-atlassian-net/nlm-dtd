source-type

Source Type

This attribute is used in two senses:

  • To identify the kind of source providing an award that sponsored a work or the research on which a work was based
  • To identify the type of publication, for example, a book, journal, database, etc., that is (or contains) the target of the <related-object>

Attribute Values

In Element

<funding-source>
Value Meaning
Text, numbers, or special characters Type of entity that is source of an award sponsoring a work or the research underlying it
Restriction: This attribute may be specified if the element is used.

In Element

<related-object>
Value Meaning
Text, numbers, or special characters Type of object that is source of the related object (<related-object>), for example, “book”
Restriction: This attribute may be specified if the element is used.
