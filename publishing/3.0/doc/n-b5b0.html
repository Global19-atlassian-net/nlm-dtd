%doc-back-matter-mix;

Document Back Matter Elements

Elements used inside the back matter (element <back>) of an article, to describe ancillary material such as an appendix

Remarks

This parameter entity (%doc-back-matter-mix;) has been significantly remodeled in version 3.0 of the Tag Set. For users who customize the Tag Set, the values and/or usage in version 3.0 are not backward-compatible with that in previous versions of the Tag Set. Specifically, in prior versions the %doc-back-matter-mix; parameter entity included the %sec-level; parameter entity, which has been replaced by the %sec.class; parameter entity. This change has no effect on compatibility unless the derivative Tag Sets have been customized.

Content Model

<!ENTITY % doc-back-matter-mix
                        "%back.class; | %front-back.class; | 
                         %sec.class;"                                >
