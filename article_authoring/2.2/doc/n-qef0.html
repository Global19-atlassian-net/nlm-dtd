%nlmcitation.ent;

NLM Citation Module

Definition

External module in the Article Authoring DTD (Pumpkin) that contains the only non-Suite-default element, the highly structured NLM citation (<nlm-citation>). This element enforces a slightly loose version of an NLM-structured bibliographic reference in which sequence of elements is enforced and interior punctuation should not be in the data because the expectation is that such punctuation will be generated. (The module %nlmcitation.ent; was initially written as part of the Journal Publishing DTD and is used without change by the Article Authoring DTD.)

Declaration

<!ENTITY % nlmcitation.ent
                        PUBLIC
"-//NLM//DTD NLM Citation v2.0 20040830//EN"
"nlmcitation.ent"                                                    >
