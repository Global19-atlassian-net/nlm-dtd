%display-back-matter.class;

Display Element Back Matter Class Elements

Definition

Miscellaneous attribution or copyright elements that may occur at the end of a <preformat> or <statement> element

Remarks

Not to be confused with the Parameter Entity %display-back-matter; which presents a structured sequence involving the <attrib> and <permissions> elements; that structured sequence is used at the end of boxed text, a figure, or similar display element.

Declaration

<!ENTITY % display-back-matter.class
                        "attrib | permissions"                       >

Module
