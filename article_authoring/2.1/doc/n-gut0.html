xlink:title

Title of the Link

Definition

Describes the function of a link’s remote resource in human-readable fashion

Remarks

This attribute’s value, i.e., “the link title”, may be displayed at the point in an online article where the link occurs. A user could read the title and determine whether to follow the link.

Attribute Values

In Elements

<abbrev> Abbreviation or Acronym; <bio> Biography; <chem-struct> Chemical Structure (Display); <citation> Citation; <collab> Collaborative (Group) Author; <contract-num> Contract/Grant Number; <contract-sponsor> Contract/Grant Sponsor; <contrib> Contributor; <email> Email Address; <ext-link> External Link; <graphic> Graphic; <inline-graphic> Graphic, Inline; <inline-supplementary-material> Inline Supplementary Material; <institution> Institution Name: in an Address; <license> License Information; <media> Media Object; <named-content> Named Special (Subject) Content; <nlm-citation> NLM Citation Model; <product> Product Information; <related-article> Related Article Information; <self-uri> URI for This Same Article Online; <supplementary-material> Supplementary Material; <uri> Uniform Resource Indicator (URI)
Value Meaning
Text, numbers, or special characters A descriptive, human-readable title that acts as a brief description of the link
Restriction: This attribute may be specified if the element is used.
