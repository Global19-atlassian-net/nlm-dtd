<contract-num>

Contract/Grant Number

Definition

Contract or grant number of the supported work described in an article

Remarks

If an acronym for the sponsor is available, insert it before the number.

Attributes

id Identifier
rid Reference to an Identifier
xlink:actuate Actuating the Link
xlink:href Href (Linking Mechanism)
xlink:role Role of the Link
xlink:show Showing the Link
xlink:title Title of the Link
xlink:type Type of Link
xmlns:xlink XLink Namespace Declaration

Model Information

Content Model

<!ELEMENT  contract-num
                        (#PCDATA %contract-elements;)*               >

Description

Text, numbers, or special characters, zero or more
