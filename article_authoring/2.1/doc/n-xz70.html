%edition-elements;

Edition Elements

Definition

Elements that can be included along with data characters inside the content model of the <edition> element

Remarks

This Parameter Entity is currently set as a null string, i.e., there is no string content. However, a customization module could redefine this Parameter Entity, thereby adding elements to the <edition> element.

Declaration

<!ENTITY % edition-elements
                        ""                                           >
