<title>

Title

Heading or title for a structural element, such as a <sec>, <app>, <boxed-text>, etc.

Attributes

content-type Type of Content
id Identifier

Related Elements

This element is not used for any article-level titles.

  • Within the metadata for the article, the titles (which are grouped in a <title-group> container element) are explicitly named: <article-title> followed by the optional article <subtitle>;
  • Within the bibliographic reference (<element-citation> or <mixed-citation>) an article-title is also <article-title>, while a book or journal title (that contains the cited article or chapter) is <source>.

Content Model

<!ELEMENT  title        (#PCDATA %struct-title-elements;)*           >

Expanded Content Model

(#PCDATA | email | ext-link | uri | bold | italic | monospace | overline | roman | sc | strike | underline | alternatives | inline-graphic | chem-struct | inline-formula | abbrev | named-content | styled-content | fn | target | xref | sub | sup | break | element-citation | mixed-citation | nlm-citation)*

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • <email> Email Address
    • <ext-link> External Link
    • <uri> Uniform Resource Indicator (URI)
  • All the emphasis elements:
    • <bold> Bold
    • <italic> Italic
    • <monospace> Monospace Text (Typewriter Text)
    • <overline> Overline
    • <roman> Roman
    • <sc> Small Caps
    • <strike> Strike Through
    • <underline> Underline
  • <alternatives> Alternatives For Processing
  • <inline-graphic> Graphic, Inline
  • <chem-struct> Chemical Structure (Display)
  • <inline-formula> Formula, Inline
  • All the inline elements used to surround “special” content:
    • <abbrev> Abbreviation or Acronym
    • <named-content> Named Special (Subject) Content
    • <styled-content> Styled Special (Subject) Content
  • All the simple link/cross-referencing elements:
    • <fn> Footnote
    • <target> Target of an Internal Link
    • <xref> X(cross) Reference
  • <sub> Subscript
  • <sup> Superscript
  • <break> Line Break
  • All the citation elements:
    • <element-citation> Element Citation
    • <mixed-citation> Mixed Citation
    • <nlm-citation> NLM Citation

This element may be contained in:

<abstract>, <ack>, <app>, <app-group>, <author-comment>, <bio>, <caption>, <def-list>, <disp-quote>, <fn-group>, <glossary>, <kwd-group>, <list>, <ref-list>, <sec>, <statement>, <supplement>, <table-wrap-foot>, <verse-group>
