Article Authoring Tag Set Tag Library version 3.0

Digital Archive of Journal Articles
National Center for Biotechnology Information (NCBI)
National Library of Medicine (NLM)

Changes by Element

This section describes, for each element in the Authoring Tag Set that changed, the differences between Version 2.3 and Version 3.0 of the Tag Set.

The Following Elements were Deleted from 3.0:

  • <chem-struct-wrapper> (renamed <chem-struct-wrap>)
  • <citation>
  • <contract-num>
  • <contract-sponsor>
  • <font>
  • <gloss-group> (renamed <glossary>)
  • <grant-num>
  • <grant-sponsor>

The Following Elements were Added in 3.0:

  • <alternatives>
  • <award-group>
  • <award-id>
  • <chapter-title>
  • <chem-struct-wrap> (formerly <chem-struct-wrapper>)
  • <compound-kwd>
  • <compound-kwd-part>
  • <date-in-citation>
  • <element-citation>
  • <funding-group>
  • <funding-source>
  • <funding-statement>
  • <issue-part>
  • <license-p>
  • <mixed-citation>
  • <open-access>
  • <part-title>
  • <principal-award-recipient>
  • <principal-investigator>
  • <related-object>
  • <roman>
  • <styled-content>
  • <textual-form>
  • <volume-series>
