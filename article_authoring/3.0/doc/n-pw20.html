<app-group>

Appendix Matter

Container element for one or more appendices (<app> elements), which hold additional or ancillary material attached to an article

Remarks

Technical Note: This element is highly similar to the Appendix Matter <appm> element used in AAP- and ISO-12083-derivative journal tag sets.

Attributes

content-type Type of Content
id Identifier
specific-use Specific Use

Content Model

<!ELEMENT  app-group    %app-group-model;                            >

Expanded Content Model

(title?, p*, (app)*)

Description

The following, in order:

  • <title> Title, zero or one
  • <p> Paragraph, zero or more
  • <app> Appendix, zero or more

This element may be contained in:

<back>

Example

<article>
<front>
...</front>
<body>...</body>
<back>
<ref-list>...
</ref-list>
<app-group>
<app><title>Appendix</title>
<sec><title>The Bipolar Seesaw.</title>
<p>The evidence for the antiphasing of the millennial-duration climate 
changes occurring on the Antarctic continent ...</p>...
</app>
</app-group>

</back>
</article>
