<label>

Label (Of a Statement)

The number and any prefix word associated with the title of a <statement>, such as “Postulate 3.” or “Theorem 2.”

Content Model

<!ELEMENT  label        (#PCDATA %label-elements;)*                  >

Expanded Content Model

(#PCDATA)*

Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<disp-formula>, <disp-formula-group>, <statement>
