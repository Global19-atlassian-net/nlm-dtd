MathML Setup Module

External module in the base Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from the Article Authoring Tag Set to set up the namespaces and parameter entities for the MathML elements

Remarks

This module then calls the MathML Tag Set to include the MathML elements in a tag set.
