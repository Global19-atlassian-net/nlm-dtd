rid

Reference to an Identifier

Points to the identifier of the associated element as a means of linkage, for example, from a <term> to a <def>, or from a <bio> to a <contrib>

Attribute Values

In Elements

<aff>, <award-group>, <bio>, <award-id>, <contrib>, <def>, <funding-source>, <funding-statement>, <term>, <xref>
Value Meaning
One or more identifiers (IDREFS) Points to one or more existing identifiers
Restriction: This attribute may be specified if the element is used.
