<date-in-citation>

Date within a Citation

Within bibliographic references (<element-citation> and <mixed-citation>), a date other than the publication date of the cited work, for example the access date, the copyright date, or a time stamp indicating when the work was published for a continuously or frequently updated source.

Remarks

For a detailed discussion on the use of <date-in-citation>, see Dates in Citations.

Attribute

content-type Type of Content

Related Elements

Within citations (<element-citation> and <mixed-citation>), this element is used to name the date of publication. The elements <year>, <day>, <month>, and <season> may all be used to describe the date a cited resource was published. Other dates inside a citation, such as a copyright date, the date on which the author accessed the resource, or a withdrawal date, should be tagged using <date-in-citation> with the @content-type attribute used to name the type of date (copyright, access-date, time-stamp, etc.).

Content Model

<!ELEMENT  date-in-citation  
                        (#PCDATA %date-in-citation-elements;)*       >

Expanded Content Model

(#PCDATA | day | month | season | year)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <day> Day
  • <month> Month
  • <season> Season
  • <year> Year

This element may be contained in:

<element-citation>, <mixed-citation>, <product>, <related-article>, <related-object>
