name

Name

This attribute is used in two senses:

  • For use within the mathematics element (<mml:math>), see the documentation for the Mathematical Markup Language (MathML) 2.0 Tag Set.
  • For the Private Character (Custom or Unicode™), where it contains the unique name for the private character being defined

Remarks

The @name attribute for a defined private character <private-char> provides a Unicode™-style name (an all uppercase name) for the private character, for example, “COMBINING CYRILLIC PALATALIZATION”. In contrast, the @description attribute for a defined private character <private-char> provides a rough visual description, for example, “combining vertical tilde”.

Attribute Values

In Element

<mml:math>
Value Meaning
Text, numbers, or special characters For usage with <mml:math>, see MathML documentation.
Restriction: This attribute may be specified if the element is used.

In Element

<private-char>
Value Meaning
Text, numbers, or special characters For usage with <private-char>, the name of the character being defined. By convention, this character should be named in a fashion similar to names found in the Unicode™ standard (e.g., in all uppercase ASCII, “NORTHWEST SOUTHEAST ARROW”).
Restriction: This attribute may be specified if the element is used.
