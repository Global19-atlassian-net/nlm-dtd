count

Count

Contains the number of (a count of) pages that are contained within a referenced item, such as a book

Attribute Values

In Element

<page-count>
Value Meaning
Number Although the Tag Set cannot limit this attribute beyond letters, numbers, and special characters, validation beyond the DTD (for example, schema validation or Schematron validation) may constrain this attribute to a counting number (positive integer), for example “15”.
Restriction: This attribute must be specified if the element is used.
