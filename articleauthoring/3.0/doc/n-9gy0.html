%oasis-tablesetup.ent;

OASIS XML Table Setup Module

External module named in the base Suite, but not used in the Article Authoring Tag Set, to set up the parameter entities needed by the OASIS (CALS) Table Exchange table model

Remarks

Authoring Note: The Article Authoring Tag Set uses the XHTML table model, which is called by the %XHTMLtablesetup.ent; module which calls the %xhtml-table-1.mod; module to add the XHTML table tags to a tag set. If the OASIS (CALS) table model is wanted, the %oasis-tablesetup.ent; module should be invoked, which calls the %oasis-exchange.ent; module to add the OASIS table tags to a tag set.
