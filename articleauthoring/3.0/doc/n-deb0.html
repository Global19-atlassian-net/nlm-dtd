Sample Citations for Authoring

This section contains samples of various types of citations tagged according to this Authoring Tag Set. Tagging citations is an editorial decision and an art, so these are not the only way the cited works could have been tagged. For many publishers, they will not even be best practice, but they show one way and may help guide tagging use. Each sample that follows can be assumed to be inside a <ref>, which is inside a <ref-list>. The text will be shown twice, the first tagged with the element-style <element-citation> element and the second with the mixed-style <mixed-citation> element.
