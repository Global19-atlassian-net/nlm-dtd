page

Page of Related Article

Page number of a related article, which may be either the number of the page on which the article starts or a page range

Usage

The @page attribute is used with the @vol, @issue, and @journal-id attributes to identify the related article <related-article>.

Attribute Values

In Element

<related-article>
Value Meaning
Text, numbers, or special characters A page number, which may be numeric such as “1145” or “35-583”; or a letter/number combination, such as “46a” or “iii”. It is best practice to make the value a first page or a page range.
Restriction: This attribute may be specified if the element is used.
