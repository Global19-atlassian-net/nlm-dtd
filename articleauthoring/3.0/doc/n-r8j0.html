%pub-id-types;

Article/Publication Identifier Types

The explicit list of values for the @pub-id-type attribute, which names the type of identifier, or the organization or system that defined the identifier, for a specific ID of a cited journal article or other work

Remarks

The @pub-id-type attribute names the type of identifier, or the organization or system that defined the identifier, for the identifier of a cited work. This attribute is associated with the <pub-id> element, which is an identifier for a publication cited in a bibliographic reference (<element-citation> or <mixed-citation>).
