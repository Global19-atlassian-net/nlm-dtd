%XHTMLtablesetup.ent;

XHTML Table Setup Module

External module in the base Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from a DTD to set up the parameter entities needed by the XHTML table model

Remarks

Authoring Note: The Article Authoring Tag Set uses the XHTML table model, which is called by the %XHTMLtablesetup.ent; module. This module then calls the %xhtml-table-1.mod; module to add the XHTML table tags to a tag set.
