<target>

Target of an Internal Link

An anchor (fixed location) that may be placed anywhere within textual material such as a paragraph to provide a location to which a cross reference can point

Remarks

This element (<target>) has been significantly remodeled in version 3.0 of the Tag Set. The values and/or usage in version 3.0 are not backward-compatible with that in previous versions of the Tag Set. Specifically, in prior versions the <target> element did not require an @id attribute.

Attributes

id Identifier
target-type Type of Target

Content Model

<!ELEMENT  target       (#PCDATA %target-elements;)*                 >

Expanded Content Model

(#PCDATA | bold | italic | monospace | overline | roman | sc | strike | underline | named-content | styled-content | sub | sup)*

Description

Any combination of:

  • Text, numbers, or special characters
  • All the emphasis elements:
    • <bold> Bold
    • <italic> Italic
    • <monospace> Monospace Text (Typewriter Text)
    • <overline> Overline
    • <roman> Roman
    • <sc> Small Caps
    • <strike> Strike Through
    • <underline> Underline
  • <named-content> Named Special (Subject) Content
  • <styled-content> Styled Special (Subject) Content
  • <sub> Subscript
  • <sup> Superscript

This element may be contained in:

<aff>, <article-title>, <attrib>, <bold>, <chem-struct>, <def-head>, <italic>, <license-p>, <monospace>, <named-content>, <overline>, <p>, <product>, <roman>, <sc>, <speaker>, <strike>, <styled-content>, <sub>, <subtitle>, <sup>, <td>, <term>, <term-head>, <th>, <title>, <trans-title>, <underline>
