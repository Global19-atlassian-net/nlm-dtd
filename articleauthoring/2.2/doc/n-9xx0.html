<title>

Title

Definition

Heading or title for a structural element, such as a <sec>, <app>, <boxed-text>, etc.

Related Elements

In spite of its name, this element is not used for any of the titles of an article:

  • Within the metadata for the article, the titles (which are grouped in a <title-group> wrapper) are explicitly named: <article-title> followed by the optional article <subtitle>;
  • Within the bibliographic reference (<citation> or <nlm-citation>) an article-title is also <article-title>, while a book or journal title (that contains the cited article or chapter) is <source>.

Model Information

Content Model

<!ELEMENT  title        (#PCDATA %struct-title-elements;)*           >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Formula, Inline <inline-formula>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Line Break <break>

This element may be contained in:

<abstract> Abstract; <ack> Acknowledgments; <app> Appendix; <app-group> Appendix Matter; <author-comment> Author Comment; <bio> Biography; <boxed-text> Boxed Text; <caption> Caption of a Figure, Table, Etc.; <def-list> Definition List; <disp-quote> Quote, Displayed; <gloss-group> Glossary Group; <glossary> Glossary Elements List; <kwd-group> Keyword Group; <list> List; <ref-list> Reference List (Bibliographic); <sec> Section; <statement> Statement, Formal; <table-wrap-foot> Table Wrapper Footer; <verse-group> Verse Form for Poetry
