%display-back-matter;

Display Element Back Matter Sequence

Definition

Miscellaneous attribution or copyright elements at the end of a display element such as a <fig> or <boxed-text>

Remarks

Not to be confused with the Parameter Entity %display-back-matter.class; which presents an OR grouping of the <attrib> and <permissions> elements; that unstructured string is used in the <preformat>element’s content model.

Declaration

<!ENTITY % display-back-matter
                        "attrib?, permissions?"                     >

Module

articleauthcustom-models.ent
