%default-mixes.ent;

Default Element Mixes Module

Definition

External module in the base DTD Suite that sets up the default Parameter Entities and element mix definitions used in content models for the base DTD Suite

Remarks

It is expected that some or all of the Parameter Entities declared in this module will be overridden in the DTD-specific custom mixes module, %articleauthcustom-mixes.ent;.

<!ENTITY % default-mixes.ent
