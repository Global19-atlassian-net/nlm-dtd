<product>

Product Information

Definition

Used as a wrapper for metadata about a product (such as a book, software package, hardware component, website, etc.) that is being reviewed

Remarks

This element should be used when the value of the article-type attribute on the element <article> is “book-review” or “product-review”.

A review of a book, journal, website, etc. should include as much information about the item being reviewed as is practical; see the tagged example below.

Attributes

product-type Type of Product
xlink:actuate Actuating the Link
xlink:href Href (Linking Mechanism)
xlink:role Role of the Link
xlink:show Showing the Link
xlink:title Title of the Link
xlink:type Type of Link
xmlns:xlink XLink Namespace Declaration

Model Information

Content Model

<!ELEMENT  product      (#PCDATA %product-elements;)*                >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Formula, Inline <inline-formula>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • Access Date for Cited Work <access-date>
  • Annotation in a Citation <annotation>
  • Article Title <article-title>
  • Collaborative (Group) Author <collab>
  • Comment in a Citation <comment>
  • Conference Date <conf-date>
  • Conference Location <conf-loc>
  • Conference Name <conf-name>
  • Day <day>
  • Edition, Cited <edition>
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • Electronic Location Identifier <elocation-id>
  • First Page <fpage>
  • Government Report, Cited <gov>
  • ISBN <isbn>
  • ISSN <issn>
  • Issue Number <issue>
  • Issue Identifier <issue-id>
  • Issue Title <issue-title>
  • Last Page <lpage>
  • Month <month>
  • Name of Person <name>
  • Object Identifier <object-id>
  • Page Count <page-count>
  • Page Ranges <page-range>
  • Patent Number, Cited <patent>
  • Person Group for a Cited Publication <person-group>
  • Publication Identifier for a Cited Publication <pub-id>
  • Publisher’s Location <publisher-loc>
  • Publisher’s Name <publisher-name>
  • Role or Function Title of Contributor <role>
  • Season <season>
  • Series <series>
  • Source <source>
  • Standard, Cited <std>
  • Supplement Information <supplement>
  • Time Stamp for Cited Work <time-stamp>
  • Translated Source <trans-source>
  • Translated Title <trans-title>
  • Volume Number <volume>
  • Volume Identifier <volume-id>
  • Year <year>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
