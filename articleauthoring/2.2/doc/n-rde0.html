%intable-para.class;

Inside a Table Paragraph Class Elements

Definition

A collection of the simpler of the paragraph-level elements named in one place for use inside a <table-wrap> element, as an alternative to a table tag

Remarks

Usually a <table-wrap> element contains a table, properly tagged with rows and columns according to the XHTML table model. But sometimes, a structure that is labeled as a “table” is actually a list, or two paragraphs, or some other non-tabular structure.

This Parameter Entity is used inside the Parameter Entity %inside-table-wrapper;, which names all the elements allowed in the middle of a <table-wrap> element.

Declaration

<!ENTITY % intable-para.class
                        "disp-quote | speech | statement |
                         verse-group"                                >

Module

default-classes.ent
