axis

Axis (XHTML table model)

Definition

See documentation for the HTML 4.0 (XHTML) table model.

A method of categorizing table cells to provide easier access for non-visual browsers

Remarks

This attribute was not widely supported as of December 2002.

Attribute Values

In Elements

<td> Table Data Cell (XHTML table model)
<th> Table Header Cell (XHTML table model)
Value Meaning
Text, numbers, or special characters Names the axis
Restriction: This attribute may be specified if the element is used.
