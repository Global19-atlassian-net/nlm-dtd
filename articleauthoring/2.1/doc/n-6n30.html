<country>

Country: in an Address

Definition

Country name used within an address, for example a postal address

Related Elements

An address (<address>) may use the following elements, in order: <institution>, <addr-line>, <country>, <phone>, <fax>, <email>, <ext-link>, and <uri>. At minimum, the element <country> must be used.

Model Information

Content Model

<!ELEMENT  country      (#PCDATA)                                    >

Description

Text, numbers, or special characters

This element may be contained in:

<address> Address/Contact Information; <aff> Affiliation
