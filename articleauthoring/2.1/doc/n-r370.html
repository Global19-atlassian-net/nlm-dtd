%block-display.class;

Block Display Class Elements

Definition

Names graphical or other image-related elements in the base DTD Suite that cannot be used inline, but only as block elements; the display elements may occur within the text of a paragraph although they are typically at the same hierarchical level as a paragraph.

Declaration

<!ENTITY % block-display.class
                        "boxed-text | chem-struct-wrapper | fig |
                         graphic | media | preformat |
                         supplementary-material | table-wrap"        >

Module

articleauthcustom-classes.ent
