alternate-form-of

Alternate Form of Graphic, Media Object, Etc.

Definition

Where multiple formats of an object (such as a graphic file, media object, or chemical structure) are available, this attribute indicates that a format is a secondary one and provides a link to the primary format, so that only one format of an item can be displayed, if desired.

Attribute Values

In Elements

<caption> Caption of a Figure, Table, Etc.; <chem-struct> Chemical Structure (Display); <col> Table Column (XHTML table model); <colgroup> Table Column Group (XHTML table model); <disp-formula> Formula, Display; <graphic> Graphic; <inline-graphic> Graphic, Inline; <inline-supplementary-material> Inline Supplementary Material; <media> Media Object; <mml:math> Math (MathML 2.0 Tag Set); <preformat> Preformatted Text; <related-article> Related Article Information; <supplementary-material> Supplementary Material; <table> Table (XHTML table model); <tbody> Table Body (XHTML table model); <td> Table Data Cell (XHTML table model); <tfoot> Table Footer (XHTML table model); <th> Table Header Cell (XHTML table model); <thead> Table Header (XHTML table model); <tr> Table Row (XHTML table model)
Value Meaning
An identifier (IDREF) The pointing uses the ID/IDREF mechanism, using this “alternate-form-of” attribute of an element to point to the “ID” attribute of an existing Graphic, Media Object, or other similar element.
Restriction: This attribute may be specified if the element is used.
