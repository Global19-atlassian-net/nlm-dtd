%ref-types;

Type of Cross(x)-Reference

Definition

Lists the values of the ref-type attribute, which is used to name the type of element to which an internal reference is pointing; for example, the value “fig” means that the reference is pointing to a <fig>.

Remarks

The values may be used to control type-specific processing or validation.

Declaration

<!ENTITY % ref-types    "aff | app | bibr |
                         boxed-text | chem | contrib | corresp |
                         disp-formula | fig | fn | kwd | list |
                         plate | scheme | sec | statement |
                         supplementary-material |
                         table | table-fn |
                         other"                                      >
