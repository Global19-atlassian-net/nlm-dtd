%anonymous-model;

Anonymous Model

Definition

Content model for the <anonymous> element

Remarks

Unlike the Archiving Tag Set, the Authoring Tag Set defines the <anonymous> element as an EMPTY element, i.e., it has no content. When present, it is used to generate the text “anonymous”, “anon.”, etc.

Declaration

<!ENTITY % anonymous-model
                        "EMPTY"                                      >

Module
