<app-group>

Appendix Matter

Definition

Wrapper element for one or more appendices (<app> elements), which hold additional or ancillary material attached to an article

Remarks

This element is the same as the Appendix Matter <appm> element used in AAP- and ISO 12083-derivative DTDs.

Attributes

content-type Type of Content
id Identifier

Model Information

Content Model

<!ELEMENT  app-group    %app-group-model;                            >

Description

The following, in order:

  • Title <title>, zero or one
  • Paragraph <p>, zero or more
  • Appendix <app>, zero or more

This element may be contained in:

<back> Back Matter
