<disp-quote>

Quote, Displayed

Definition

Extract or extended quoted passage from another work, usually made typographically distinct from surrounding text

Remarks

Use this element for epigraphs, as well as for block quotes and extracts within text.

Attributes

content-type Type of Content
id Identifier
xml:lang Language

Model Information

Content Model

<!ELEMENT  disp-quote   %disp-quote-model;                           >

Description

The following, in order:

  • Title <title>, zero or one
  • Paragraph <p>, one or more
  • Attribution <attrib>, zero or one
  • Permissions <permissions>, zero or one

This element may be contained in:

<app> Appendix; <body> Body of the Article; <boxed-text> Boxed Text; <fig> Figure; <named-content> Named Special (Subject) Content; <p> Paragraph; <sec> Section; <table-wrap> Table Wrapper
