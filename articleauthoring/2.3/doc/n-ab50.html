<grant-sponsor>

Grant Sponsor

Definition

Name of the organization that awarded the grant supporting the work described in the article

Remarks

Best Practice: If an acronym for the grant sponsor is available, insert it before the number in the <grant-num>. The full name of the grant sponsor can be captured in the <grant-sponsor> element.

Attributes

content-type Type of Content
id Identifier
rid Reference to an Identifier

Model Information

Content Model

<!ELEMENT  grant-sponsor
                        (#PCDATA %grant-sponsor-elements;)*          >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
  • Graphic, Inline <inline-graphic>
  • Formula, Inline <inline-formula>
  • Abbreviation or Acronym <abbrev>
  • Named Special (Subject) Content <named-content>
  • Subscript <sub>
  • Superscript <sup>

This element may be contained in:

<article-meta> Article Metadata; <p> Paragraph
